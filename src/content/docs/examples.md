# Factory Examples

Explore ready-to-use factory configurations that demonstrate how to compose skills, commands, agents, and workflows for common development scenarios.

## Available Examples

### Documentation Factory
- [Simple Documentation Factory](/examples/01-simple-documentation-factory) - A minimal factory setup for generating and maintaining project documentation

## What is a Factory Example?

A factory example is a complete, working configuration that combines multiple components (skills, commands, agents, workflows) to accomplish a specific development goal. Each example includes:

- **Component list**: Which tools are used
- **Configuration**: How the components are set up
- **Use cases**: What problems the factory solves
- **Setup instructions**: How to get started

## Using Examples

1. Choose an example that matches your needs
2. Install the required skills and tools
3. Customize the configuration for your project
4. Run the factory workflow

Examples are designed to be starting points. Feel free to modify, extend, or combine them to create your ideal development environment.
